-----

# How to play

```bash
python opaseety.py 
```

# Contributing

Requires [hatch](https://hatch.pypa.io/latest/), a modern Python project manager.

```
git clone 
```

```
cd opaseety
```

```
hatch shell
```

Run example with 

```
python .\opaseety\opaseety.py -i .\data\
```
